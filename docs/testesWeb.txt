O selenium usa um Driver para se comunicar com o browser
Código (Selenium)-----> Driver-----> Browser

Existe um Driver para cadqa versão ou conjunto de versões de um navegador
No caso Chrome/Edge: 1 para cada versão
Nos demais normalmente 1 versão de driver funciona para várias do navegsador

Hay 3 formas de configurar/utilizar um Driver:
- Download manual e instala dentro do proyeto - bom (Potrtatil)- Ruim: Manuntenção
- Download manual e instala em uma pasta compartilhada - Bom : Manuntenção - Ruim Portatil
-Gerenciador de download automático - Bom: Manuntenção + Portátil- Ruim: Usa recursos
